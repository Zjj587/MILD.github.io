MILD

A Manipulation Interface Localization Dataset for UMI-Style Robot Teaching

MILD is designed for studying visual-inertial localization under manipulation-driven motion: handheld teaching, object interaction, low texture surfaces, occlusion, lighting changes, and long-horizon task execution.

Why MILD

Localization where manipulation breaks the easy assumptions.

UMI-style teaching creates camera and IMU trajectories that differ from standard walking, driving, or drone datasets. The motion is close to objects, frequently occluded by hands and task objects, and often includes fast rotations, pauses, contact-rich manipulation, and difficult visual texture.

The website is structured around release-facing task families first: every task has a captured preview image, scene-variant summary, sensor availability, and a dataset link location ready for release.

The current snapshot contains manipulation, wiping, bookshelf, box, and motion-pattern scenes. The task explorer presents them as benchmark scenarios instead of temporary collection folder names.

Sequences & Variants

One task scene expands into many localization sequences.

For each operation scene, MILD separates the table capture from tag-assisted captures. The collection uses table, ArUco, AprilTag Custom48h12, and appearance-variant folders, with four-tag layouts as the primary marker condition and selected one- or two-tag subsets for sensitivity studies.

This organization keeps the benchmark useful for both pure visual-inertial localization and marker-aided localization analysis without mixing the conditions.
